Progressive Web Application (PWA)

PWA is a web-app that can operate both as a web-page and mobile-app.
They can be installed without an app store. On ios find the share button then choose "add to home screen".


In order to function like an APP A "manifest.json" file is added that gives the webpage an app icon, a start point, and instructions on how to work offline.
